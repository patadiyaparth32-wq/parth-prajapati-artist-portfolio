# Component Documentation

## Navbar
- **Purpose:** Responsive header with links to sections.
- **Features:** Glassmorphism backdrop, theme toggle, and auto-hide on scroll.

## Hero
- **Purpose:** Introduces the artist and highlights a featured masterpiece.
- **Animations:** Floating artwork frame and parallax scroll effects.

## Gallery
- **Purpose:** Grid displaying drawings with categories and detail cards.
- **Filters:** `ALL`, `MASTERPIECES`, `PENCIL`, `COLOUR`, `EXHIBITIONS`.
- **Interactivity:** Interactive lightbox with zoom, panning, and details.

## Testimonials
- **Purpose:** Autoplay review slider for collectors and students.
- **Features:** Swipe gestures on mobile, keyboard controls, and pause on hover.

## Contact
- **Purpose:** Validated form that sends messages to the database.
- **Design:** Reusable form fields with floating labels and error states.

## Back to Top
- **Purpose:** Float button to scroll to the top.
- **Behavior:** Visible after scrolling 500px, with hover glow effects.
