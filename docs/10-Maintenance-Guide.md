# Maintenance Guide

Best practices for keeping the website secure and updated.

## Regular Maintenance Checklist

### 1. Update Dependencies
Check for and update packages periodically:
```bash
npm outdated
npm update
```

### 2. Manage Artworks
Ensure new images added to the gallery are optimized:
- **Format:** WebP or optimized JPEG.
- **File Size:** Under 300KB.
- **Dimensions:** Max width of 1920px.

### 3. Backup Leads
Periodically export the Google Sheet database to a CSV file for backup.
