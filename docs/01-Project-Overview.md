# Project Overview

This website is a luxury digital gallery and portfolio built for **Parth Prajapati**, a professional drawing artist and educator.

## Objectives
- **Exhibition Showcase:** Present fine art drawings (charcoal, graphite, color) in an online gallery.
- **Lead Generation:** Capture inquiries for commissions, purchases, and art classes.
- **Mentorship Registration:** Highlight educational courses and acquire student leads.

## Key Features
- **Visuals:** Framed artwork presentations, parallax scrolling, and smooth entry animations.
- **Responsive Navigation:** Accessible navigation with smooth scrolling and category-aware observers.
- **Bilingual Theme:** Luxury dark mode and warm light mode that automatically adapt to system settings.
- **CORS-Free Submission:** Contact form connected to Google Sheets via a Next.js server-side API route.
