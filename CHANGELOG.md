# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-29

### Added
- **Cinematic Hero Section:** Added a premium framed artwork display with a floating details card, featuring parallex scrolling, hover-scale reveals, and floating animation.
- **Dynamic Gallery Grid:** Developed a categorised gallery supporting filters (`ALL`, `MASTERPIECES`, `PENCIL`, `COLOUR`, `EXHIBITIONS`).
- **Interactive Lightbox:** Built a custom media viewer displaying high-resolution artwork images, descriptions, categories, and tags, with keyboard and swipe controls.
- **Testimonials Slider:** Created an autoplaying review slider alternating between `Collector` and `Student` categories, featuring Studio Ghibli-inspired WebP avatars.
- **Next.js Server API Proxy:** Added `/api/contact` to proxy contact submissions to Google Apps Script, bypassing browser CORS preflight restrictions.
- **Contact Form UI/UX:** Refactored the form with a reusable `FormField` component, GPU-accelerated floating labels, a no-shift error layout, and Chrome autofill styling overrides.
- **Back to Top Button:** Implemented a floating glassmorphic button with a 500px scroll threshold, gold borders, and custom hover glows.
- **Theme System:** Configured a tailwind-compatible theme system with system detection and persistence for dark and light modes.
- **Developer Credit Bar:** Restored the footer credit bar for GM Web Studio.
